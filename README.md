# MSAL-YOLO
 A Multi-Strategy Collaborative Enhancement Framework for Robust Small and Dense Object Detection in UAV Aerial Imagery


# Highlights

- Spatial–Channel Mixed Convolution (SCMConv)
- Multi-Branch Enhanced Coordinate Attention (MBECA)
- Lightweight and efficient design
- Improved small-object detection performance

## Results on VisDrone2019

| Model           | mAP@0.5 | mAP@0.5:0.95 |
| YOLOv8 Baseline | 30.0    |     17.0     |
| MSAL-YOLO       | 35.7    |     20.5     |




This repository provides the official implementation of MSAL-YOLO (*Manuscript under review*).

It includes:
- Network architecture configurations  
- Core module implementations (SCMConv, MBECA, RA-WIoU)  
- Related experimental files and setups
